# Cinema Seat Booking System 🎬

> An interactive movie seat booking system with real-time availability tracking and dynamic pricing.

![Project Preview](project3-cinema.png)

## 🔗 Live Demo
[View Project](https://dulcet-pavlova-a0e8b3.netlify.app/)

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript
- Local Storage

## ✨ Features
- Interactive seat selection
- Real-time seat availability tracking
- Dynamic pricing calculation
- Movie selection dropdown
- Occupied/available/selected seat indicators
- Data saved with Local Storage

